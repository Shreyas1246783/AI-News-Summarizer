# AI-News-Summarizer
This project is an automated n8n workflow that scrapes the latest Artificial Intelligence news from multiple RSS feeds, uses Google Gemini to synthesize and summarize the content, and sends a curated daily digest via Gmail.

In this project, let's enhance  AI News Summarizer workflow by adding real-time event updates to give us a complete view of what's happening in the AI world. This workflow automatically gathers the latest AI news from RSS feeds, fetch Live Events from Google Search results, uses a generative AI to create a structured daily report, and emails it to you.
